Airline Booking & Flight Management System

Overview

A MySQL-based DBMS project to manage airports, flights, passengers, bookings, payments, and airline operations.

Features

- Flight scheduling
- Passenger management
- Ticket booking and cancellation
- Payment management
- Crew management

SQL Concepts Used

- DDL & DML Commands
- Primary and Foreign Keys
- Joins
- Subqueries
- Views
- Stored Procedures
- Triggers

Technologies

- MySQL
- SQL
- DBMS
